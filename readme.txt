Плагин быстро установливает на ваш сайт виджет нашего сервиса [Jobs Widget](https://jobswidget.me "").  JobsWidget сервис дает возможность быстро отобразить красивый список вакансий вашей компании на любой странице корпоративного сайта и позволяет HR менеджерам самим управлять публикацией новых вакансий.


== Description ==
JobsWidget - сервис для быстрого создания страницы с вакансиями вашей компании на корпоративном сайте. Получи дополнительный эффективный источник лояльных кандидатов.  Ваши HR менеджеры смогут сами управлять размещением вакансий, без знакомства с техническими особенностями сайта.

Сервис состоит из двух частей - виджета который устанавливается на сайт при помощи данного плагина, и онлайн интерфейса через который HR менеджеры управляют публикацией новых вакансий.

Наш сервис позволяет:
* Легко создать страницу с вакансиями на корпоративном сайте.
* Импортировать описания вакансий из hh.
* Потенциальному кандидату откликнуться сразу в карточке вакансии. Ему не нужно писать письмо на адрес типа job@youcompanyname.com как это обычно принято на подобных страницах.
* Гибко настраивать внешний вид виджета, набор полей, которые используются для описания вакансии.
* Настраивать набор полей которые нужно заполнить кандидату для отклика.
* Создать для каждой вакансии список вопросов, на которые кандидат должен ответить чтобы откликнуться. Так вы можете сразу отсеивать неподходящих кандидатов.
* Настроить уведомления о новых откликах на почту.
* Следить за всеми откликами в нашем онлайн сервисе.  Чтобы понимать на какой стадии находиться общение с кандидатом, мы сделали спецуиальные статусы отражающие этапы, через которые проходит кандидат: телефонное интервью, собеседование, приглашение на работу. Статусы настраиваются в зависимости от вакансии и принятого в компании процесса найма.

== Installation ==

1.Поместите папку с плагином в [wordpress_dir] / wp-content / plugins /
2.Зайдите в интерфейс администратора WordPress и активируйте плагин
3. На странице где вы хотите разместить виджет вакансий вставьте шорт код вида [Jobs Widget 'ваш_id_в_сервисе_Jobs_Widget']. Чтобы получить свой id нужно зарегестрироваться в сервисе по ссылке [JobsWidget](https://jobswidget.me "")

== Frequently Asked Questions ==
= Какую проблему решает сервис? =
На многих корпоративных сайтах страница "Вакансии" показывает давно неактуальный список вакансий, так как HR менеджеру чтобы обновить список, нужно преодолеть большое количество разных припятствий. В частности постоянно занятого администратора сайта, подачу заявок на обновление и т.д.
Наш сервис позволяет HR менеджерам напрямую, за время чуть большее чем одна минута, самим публиковать новые вакансии на сайте. При этом им не нужно разбираться в технических подробностях. Сервис очень прост в освоении.

= В чем главное приемущество вашего сервиса? =

Сервис Job Widget позволяет HR специалистам самим управлять размещеним вакансий на сайте. Им больше не нужно обращаться для этого к администратору сайта.
Теперь список вакансий будет всегда актуален.


=Ваш сервис платный?  =
